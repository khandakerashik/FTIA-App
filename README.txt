FTIA Debian App Bundle

Contents:
- ftia-gui_1.0.0_amd64.deb: Debian package
- ftia-gui/: Self-contained frozen app bundle
- dependencies.txt: External Debian runtime packages expected by the .deb
- install.sh: Helper installer command

Install:
1. Open a terminal in this folder.
2. Run: ./install.sh

Direct run without installing:
- ./ftia-gui/ftia-gui
